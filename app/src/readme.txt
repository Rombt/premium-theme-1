Description
-------------------------------------------
Premium Theme 1 is a modern WordPress theme for corporate websites and business companies.
The theme has an adaptive design, convenient settings and support for popular libraries, which makes it a
universal solution for creating a professional website of any corporate focus.

Installation
-------------------------------------------
1. Upload the `premium-theme-1` folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance → Themes' menu in WordPress.
3. Customize the theme settings via 'Appearance → Customize'.
